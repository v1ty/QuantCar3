# Quarter-Car Suspension Modeling and Simulation

## Project Summary

This project develops a quarter-car suspension model using MATLAB,
Simulink, Simscape, and Simscape Multibody.

The model represents one vehicle corner using:

- Sprung mass
- Unsprung mass
- Suspension spring
- Suspension damper
- Tire compliance
- Road displacement input

The suspension will be tested using multiple road profiles. Performance
will be measured using ride comfort, suspension travel, and tire
deflection metrics.

## Project Goals

1. Build a working quarter-car suspension model.
